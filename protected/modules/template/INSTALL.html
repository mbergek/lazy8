Requirements

1. You must have Lazy8Web 02.00 or later installed and running properly. Preferably you should have at least a few weeks experience using Lazy8Web as well.

How to install

1. Unzip templates.lazy8web.01.00.zip into the Lazy8Web directory [web root]/lazy8/protected/modules. When you are done unzipping this then you should have the directory system [web root]/lazy8/protected/modules/template.

2. In your configuration file [web root]/lazy8/protected/config/main.config you might need to change the following at the bottom of the file if it does not already exist there:

Before

	// application-level parameters that can be accessed
	// using Yii::app()->params['paramName']
	'params'=>array(
	),
);

After

	// application-level parameters that can be accessed
	// using Yii::app()->params['paramName']
	'params'=>array(
	),
	//The following modules may or may not exist
	'modules'=>array(
		'template',
		'billing',
		'inventory',
	),
);

Thats it. Start lazy8 web and under the Setup and Data entry menus you will find the templates menus
