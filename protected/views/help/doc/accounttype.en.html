Account type data entry form
Here is where you enter your Account types. This is a screen for only advanced users who understand how the account type relates to the account and the balance/result report. Most users will probably never need to do anything here and most users would be better off never using this screen. The default records for this screen are the account types "assets, debts, incomes, expenses" or for those living in North America, "assets, debts, equity, incomes, expenses" (equity specifically added for North America). All given/added/default account types are summerized in the Balance/Result report. What you add here will effect what choices you have for "account type" in the Account Entry Window.

Company

Choose here which company you want to work with.  Most users will have only one company to work with and in such a case then there should be no selection necessary and the company will come here automatically and in such a case this field will even be disabled so you wont be able to choose a company.  Please see SETUP  for more information on how to choose a default company and Company to see how to enter more companies.
 

Account type number

This is an integer number that is automatically added to the form. You may not edit this number manually. It is uniq for just this account type you are entering for this particular company.

Account type name

This will be the name of the account type that is the default name that will show up in all reports for this account type. However, for the default account types of "assets, debts, equity (North America only), incomes, expenses", they are translated in another way into the local language for the reports. To change whether your default account types are North American or not, see the menu item "Options" and the tabwindow "Warnings" and the item called Use the equity account type When you add your own names here for new account types, use only names that you want to show up in your reports (i.e. your local language) because they will not be translated.

Sorting order for reports

It is very important that you enter the right number here. The number is only used to determine how to sort the accounts in your reports. The sorting order is normally the following:
  • "Include in the balance in reports" field descending
  • "Sorting order for reports" field
  • Account number

Include in the balance in reports

All accounts must fall into one of the following two catagories:
  • The balance report. This report contains normally assets, debts and equity. If you add any sub-catagories to these account types then you need to check this checkbox to include these in the balance report
  • The result report. This report contains normally incomes and expenses. If you add any sub-catagories to these account types then you need to uncheck this checkbox to include these in the result report
