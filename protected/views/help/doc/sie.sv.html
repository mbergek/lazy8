Svenskt Standardiserad Import och Export (SIE)
Med SIE kan du lätt flytta data från ditt ekonomisystem av ett märke till ett skatt- och deklarationssystem av ett annat. Eller till ett revisionssystem från en tredje leverantör, eller varför inte ett analysverktyg från en fjärde? Och kanske någon annan borde göra bokslutet åt dig i ett femte system?

Och varför skulle det vara annorlunda? Uppgifterna finns ju där i datorn och då fattas det väl bara att du kan låta det flyga fritt mellan dina system när du så önskar. Det är vad du kan göra med SIE. SIE står för Standardiserad Import och Export och hos http://www.sie.se kan du läsa mer om det.

Detta är ett svenskt system och inte använt till system utanför Sverige.
