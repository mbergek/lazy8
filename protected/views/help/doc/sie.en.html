Swedish Standard Import Export (SIE)
Using the export/import SIE menus, you may export/import all your balances and transactions in the SIE format which is a swedish standard and probably not useful anywhere else in the world. This format is very well defined and makes it very easy to import/export any type of book-keeping information. You may read more about this format in swedish at the http://www.sie.se
