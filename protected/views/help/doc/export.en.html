Export to CSV(Comma Separated Variable) format
This section describes how to transfer all your transactions, account schema, or Customer/Project definitions to another book-keeping system or database. This CSV export format is a universal means for transferring data. You may also want to export your information in this format and then take a normal word processor to edit the information, and then re import it again back into Lazy8Leger. That way you can do massive "search and replace" operations quickly and easily on your exported CSV file that would otherwise be impossible to do with only the Lazy8Ledger program.
When you first start this procedure, you will be presented with a dialog which will allow you to select the CSV file you want to create. This file may or may not already exist. If it already exists, it will be erased.

Selecting fields to export

Here is a list box of all the available fields to export for your table. When you first start, a minimum suggested list of fields are already chosen for you as default. These default fields represent critical data needed by most programs. You may choose any combination of fields to export. The following describes 3 buttons to help you in the selection process.

Options and field formats

There are a lot of special options and field formats available to specialize the format of your CSV file. Hit the following "Options and field formats" link to read more about it.

Default button

Hitting this button will restore the selection of fields to the minimum suggested list of fields

Select All button

Hitting this button selects all the fields in the list.

Unselect All button

Hitting this button will remove the check on all the fields in the list.

Help button

Show the help screen associated with this form.

Export button

Hitting this button creates the CSV file chosen when you first started the export procedure. The created file will contain all the fields you have chosen.

Exit button

Hitting this button closes this export window..
