Database connection

Most of the data here should not need to be ever changed.  Advanced users with a lot of programming experience might want to play with these parameters to connect to other databases or use other drivers.  If you are worried about security and want to change your password/user name you will need to come here as well

JDBC Driver information

For this release of this program this should be

com.mckoi.JDBCDriver

an nothing else.  If you are an advanced programmer you might want to try other drivers but it is really a waste of time.

Database connection

This is always empty.

User name

Here is the user name that will be used with database.  If you did not choose a user name when you installed this program, then you dont need to enter a user name here.

Password

Here is the password that will be used with database.  If you did not choose a password when you installed this program, then you dont need to enter a password here.

Connect to database

This is what you press after you have made any changes here.  This will test to see if your entries are valid.

OK

When you press this button, you will both connect to the database and exit this window. If the connection to the database fails, you will not exit the window.

Help

You get a help screen showing this help page.
