Välkommen till Lazy 8 Bokföring 2.24
Ni borde tillåta detta fönster att stanna kvar tills ni har bekantat er med Lazy 8 Bokföring. Detta fönster ger er viktig information om varje fält ni matar in information. Textrapporterna dyker upp i det tomma editorfönstret i mitten under olika flikar. Så kan man också använda den som en texteditor för att skriva, spara och öppna anteckningar och filer. Fönstren i programmet avgränsas med spalterna som man kan klicka på och dra åt olika håll. Det finns också en inställning som gör att alla fönster är fristående.
För allmän information om Lazy 8 Bokföring, klicka musen på den blåa texten 'allmän information.' Ni kan klicka på nästen varje blå text för att få mer information om detta ämnet. Om detta fönster är för litet för informationen du tittar på kan du ta tag i sidan och dra ut fönstret.
Klicka här för att se innehållsförteckning.
