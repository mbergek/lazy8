Installation

Installation by nature is a pain in the neck and you can surely make a lot of mistakes.  Please email me at support@lazy8.nu if you dont succeed.  Perhaps I can help.

CD installation for Microsoft Windows

This is by far the easiest and surest way to install onto Microsoft Windows. 
1. Click (perhaps double click) on the j2re-1_4_0-win-i.exe file on the CD in order to start the program. You will be guided through an installation of Sun microsystems Java.
2. Click (perhaps double click) on the jedit40install.jar file on the CD in order to start the program. This will install jEdit and Lazy8 Ledger.
3. Click (perhaps double click) on the jEdit icon which should be on your desktop screen in order to start the program. If you can't find it there, check in your 'Start up' menu. It is there as well. The first time you start jEdit, you will be guided through several questions. Take your time and answer carefully.
4. Thats all, you are finished.

Download installation for all operating systems.

1. This is very similar to the CD installation except that you must download from internet the programs first before running them.
2. Go to the following link http://java.sun.com/j2se/1.4/download.html and select the download from column 'JRE' and row 'Windows (all languages)' (if you are useing Microsoft Windows operating system. Otherwise choose another operating system row). Click (perhaps double click) on the j2re-1_4_0-win-i.exe file you just downloaded in order to start the program. You will be guided through an installation of Sun microsystems Java.
3. Go to the following link http://www.lazy8.nu/download/jedit40install.jar and you will immediately begin downloading jEdit and Lazy8Ledger. Click (perhaps double click) on the jedit40install.jar file you just downloaded in order to start the program. This will install jEdit and Lazy8 Ledger.
4. Click (perhaps double click) on the jEdit icon which should be on your desktop screen in order to start the program. If you can't find it there, check in your 'Start up' menu. It is there as well. The first time you start jEdit, you will be guided through several questions. Take your time and answer carefully.
5. Thats all, you are finished.
6. Just some notes for those who have another operating system then Microsoft windows. In step 2 you will get a file with a different name. Follow Sun microsystems installation instructions. In step 3, in order to run the jedit40install.jar program, you probably need to run the following command line instruction:

{path to java}/java -jar jedit40install.jar

In order to run jEdit simply type jedit at the command line.
