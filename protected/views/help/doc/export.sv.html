Exportera till CSV(Comma Separated Variabel) format
Detta beskriver hur man exportera alla verifikationer eller konto definitioner eller kund/projekt definitioner till en annan bokföring system eller databas. Detta CSV format är en universal teknik för att överföra data, fast, i Sverige finns en mycket enklare och säkrare sätt att överföra data mellan bokförings-system och det kallas SIE format. Se SIE format för mer information om SIE.

Man kan också använda CSV exportera/importera tekniken för att exportera från lazy8 bokföring, sedan manuellt ändra på data med en ordbehandlare, och sedan importera igen. På detta sätt kan man göra massiva och kraftfulla "sök och ersätta" operationer på hela databasen som är omöjligt att göra enbart i lazy8 bokföring
I början av proceduren dyker upp en dialogruta som kräver fil namnet av CSV filen som ska exporteras. Om filen redan finns, suddas den ut.

Välja fält att exportera

Här är en lista av alla tillgängliga fält att exportera för denna tabell. I början, de vanligaste fälten är ikryssat som standard. Dessa standard fält är de viktigaste fält som de flesta andra bokförings system kräver. Fast, man kan välja precis vilka fält som helst för att exportera. De följande 3 knappar snabbar upp urvals processen.

Inställningar och fält format

Det finns många inställningar och fält format för att specificera CSV formatet. Trycka på den följande "Inställningar och fält format" länk för att läsa mer om detta.

Standard urval knapp

Tryck på denna knapp för att få det standard urvalet som är de viktigaste fälten som de flesta andra bokförings system kräver.

Välja alla knappen

Tryck på denna knapp för att välja alla fält i listan.

Välja bort alla knappen

Tryck på denna knapp för att välja bort alla fält i listan.

Hjälp knappen

Visa detta hjälp fönster.

Exportera knappen

Tryck på denna knapp för att skapa CSV filen som specificerades i dialogen. Filen kommer att innehålla alla fält som du har valt.

Avsluta knappen

Tryck på denna knapp för att stänga fönstret.
