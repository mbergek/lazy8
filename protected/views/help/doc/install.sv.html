Installation

Att installera är alltid svårt, men om du har problem kan du kontakta mig email till support@lazy8.nu

CD installation för Microsoft Windows

Detta är det lättaste sättet att installera på Microsoft Windows
1. Klicka (kanske dubbelklicka) på filen j2re-1_4_0-win-i.exe som finns på CD:n för att starta programmet. Du kommer att handledas igenom installationen av Sun microsystems Java
2. Klicka (kanske dubbelklicka) på filen jedit40install.jar som finns på CD:n för att starta programmet. Du kommer att handledas igenom installationen av jEdit and Lazy8 Ledger.
3. Klicka (kanske dubbelklicka) på jEdit ikonen som ska finnas på din skrivbordsskärm. Om du inte hittar den, kolla 'start' menyn. Den finns där också. Den första gången du startar jEdit, kommer du att handledas igenom en mycket detaljerad installation av Lazy8 Ledger. Ta lite tid på dig för att svara riktigt på alla frågor.
4. Det var allt. Nu är det klart.

Internet nerladdningsinstallation för alla operativsystem.

1. Detta liknar väldigt mycket CD installationen förutom att du måste först i varje steg, ladda ner programmet innan du kan köra det.
2. Surfa till följande länk http://java.sun.com/j2se/1.4/download.html och välja nerladdning ifrån kolumnen 'JRE' och raden 'Windows (all languages)' (om du har ett annat operativsystem än Windows, välj detta operativsystems rad). Klicka (kanske dubbelklicka) på filen j2re-1_4_0-win-i.exe som du just har laddat ner för att starta programmet. Du kommer att handledas igenom installationen av Sun microsystems Java
3. Surfa till följande länk http://www.lazy8.nu/download/jedit40install.jar och du kommer omedelbart att börja att ladda ner jEdit och Lazy8Ledger. Klicka (kanske dubbelklicka) på filen jedit40install.jar som du just har laddat ner för att starta programmet. Du kommer att handledas igenom installationen av jEdit and Lazy8 Ledger.
4. Klicka (kanske dubbelklicka) på jEdit ikonen som ska finnas på din skrivbordsskärm. Om du inte hittar det, kolla då 'start' menyn. Den finns där också. Den första gången du startar jEdit, kommer du att handledas igenom en mycket detaljerad installation av Lazy8 Ledger. Ta lite tid på dig för att svara riktigt på alla frågor.
5. Det var allt. Nu är det klart.
6. Bara lite anteckningar för de som har ett annat operativsystem än Windows. I steg 2 ska du få en fil som heter något annat. Följa Sun microsystems installation instruktioner. I steg 3, för att köra jedit40install.jar programmet, måste du nog köra följande kommandorad instruktion:

{path to java}/java -jar jedit40install.jar

För att sedan köra jEdit, bara skriva jedit på kommandoraden.
