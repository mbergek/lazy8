The Lazy8 Ledger Tutorial

Lesson: The main window

The main window is what you will see when you start Lazy8 Ledger. This tutorial is only valid if you have chosen to have the 'Accountant only menus' when you started Lazy8 Ledger. Otherwise, you must go to the options, choose 'Accountant only menus', and then restart Lazy8 Ledger.
